# Animal Care Oman Web Application

A community-focused web application that provides a directory of veterinary clinics in Oman, along with FAQs, informational resources, and an integrated AI chatbot to assist pet owners in accessing veterinary services and relevant guidance.

## Technologies Used

* Python
* Django
* HTML
* CSS
* JavaScript

## Features

* Directory of veterinary clinics in Oman
* FAQ section for common pet care questions
* Informational resources for pet owners
* AI-powered chatbot for user assistance and queries
* User-friendly interface for easy navigation

## Purpose

This project was built as a community initiative to make it easier for users in Oman to find nearby veterinary clinics and access pet care information in one place, with additional AI-based assistance for improved user support.
