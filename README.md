# Veterinary Clinics Directory (Oman)

A community-focused web application that provides a directory of veterinary clinics in Oman, along with FAQs and informational resources to help pet owners access relevant veterinary services and guidance.

## Technologies Used

* Python
* Django
* HTML
* CSS

## Features

* Directory of veterinary clinics in Oman
* FAQ section for common pet care questions
* Informational resources for pet owners
* User-friendly interface for easy navigation

## Purpose

This project was built as a community initiative to make it easier for users in Oman to find nearby veterinary clinics and access basic pet care information in one place.
